# Menu Entry Manager

A TUI (Text User Interface) application for managing Linux desktop files (.desktop).

## Installation

```bash
pip install -e .
```

## Usage

```bash
menuentry
```

## Features

- View all system and user desktop entries
- Create new desktop entries
- Edit existing entries
- Set environment variables for applications
- Delete user entries

## Keybindings

- `q` - Quit
- `n` - New entry
- `s` - Save
- `d` - Delete
- `r` - Refresh list
- `j/k` - Navigate list
